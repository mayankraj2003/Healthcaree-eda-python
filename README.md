# 🏥 Healthcare Data Analysis using Python

## 📊 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on healthcare data to uncover meaningful insights about patients, medical conditions, and billing trends.

---

## 🧠 Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📂 Dataset Features

* Age
* Gender
* Medical Condition
* Billing Amount
* Admission Type

---

## 🔍 Key Analysis

* Data Cleaning (handled missing values)
* Age distribution of patients
* Gender-based analysis
* Most common medical conditions
* Billing trends analysis
